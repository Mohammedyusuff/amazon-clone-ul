# Amazon Clone UI

A responsive Amazon homepage clone built using HTML, CSS, and JavaScript. This project recreates the core user interface of the Amazon website, featuring a modern layout, responsive navigation, product sections, and interactive design while following front-end development best practices.

## Features

- Responsive and mobile-friendly design
- Amazon-inspired navigation bar
- Hero banner and product sections
- Interactive UI components
- Clean and organized code structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
